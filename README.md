Predador

Landing Page sobre "O Predador"

Descrição
Este projeto é uma landing page dedicada ao personagem e a saga Predador. O objetivo é apresentar informações sobre o personagem, filmes, onde assistir e curiosidades, com design responsivo e interativo.

Estrutura do Projeto:

index.html: Página principal com as seções do site.

src/styles/: Arquivos SCSS para estilização das seções.

src/images/: Imagens utilizadas no site.

src/scripts/: Scripts JS para interatividade.

Principais Seções

Hero: Destaque visual do personagem.

Sobre: Informações gerais sobre o personagem.

Shows: Navegação entre personagens e filmes.

Onde assistir: Plataformas disponíveis.

FAQ: Curiosidade sobre o filme.

Footer: Rodapé com créditos.

Tecnologias Utilizadas

HTML5

SCSS (Sass)

JavaScript

Gulp (automatização de tarefas)

Como rodar o projeto:

git clone https://github.com/NathanaelZampier/Projeto_Predador

Instale as dependências:

npm install

Execute o Gulp para compilar os arquivos:

gulp

Abra o arquivo index.html no navegador.
